# Maze-Game
This is a maze game with a shortest path recursive function, that not only finds the shortest path, But also ensures the maze has a valid path. Players are denoted by X (starting from (0,0) and have to reach the final destination Y (row - 1, column - 1).  After the game ends, it tells the player whether the path they travel was the shortest or not.

# How to Play?
1. Enter the number of rows and columns of the Maze.
2. Press j to move left, l to move right, i to move up and k to move down.
3. You can q to exit the game.

# Technologies / Language Used
1. The whole code is written in C++.
2. The use of dynamic memory to make the game more efficient.
